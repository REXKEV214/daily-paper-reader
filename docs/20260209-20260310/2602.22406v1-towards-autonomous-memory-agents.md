## 摘要
最近的记忆智能体通过将经验和对话历史提取到外部存储中来改进大语言模型（LLMs）。这实现了低开销的上下文组装和在线记忆更新，而无需昂贵的 LLM 训练。然而，现有的解决方案仍然是消极且被动的；记忆的增长受限于偶然获得的信息，而记忆智能体在面临不确定性时很少寻求外部输入。我们提出了自主记忆智能体，能够以最低成本主动获取、验证和整理知识。U-Mem 通过以下方式实现这一构想：(i) 一种成本感知的知识提取级联机制，从廉价的自我/教师信号逐步升级到工具验证的研究，并在必要时寻求专家反馈；(ii) 语义感知的汤普森采样（Thompson sampling），用于平衡记忆的探索与利用，并缓解冷启动偏差。在可验证和不可验证的基准测试中，U-Mem 始终优于先前的记忆基准，并能超越基于强化学习（RL）的优化，在 HotpotQA (Qwen2.5-7B) 上提升了 14.6 分，在 AIME25 (Gemini-2.5-flash) 上提升了 7.33 分。

## Abstract
Recent memory agents improve LLMs by extracting experiences and conversation history into an external storage. This enables low-overhead context assembly and online memory update without expensive LLM training. However, existing solutions remain passive and reactive; memory growth is bounded by information that happens to be available, while memory agents seldom seek external inputs in uncertainties. We propose autonomous memory agents that actively acquire, validate, and curate knowledge at a minimum cost. U-Mem materializes this idea via (i) a cost-aware knowledge-extraction cascade that escalates from cheap self/teacher signals to tool-verified research and, only when needed, expert feedback, and (ii) semantic-aware Thompson sampling to balance exploration and exploitation over memories and mitigate cold-start bias. On both verifiable and non-verifiable benchmarks, U-Mem consistently beats prior memory baselines and can surpass RL-based optimization, improving HotpotQA (Qwen2.5-7B) by 14.6 points and AIME25 (Gemini-2.5-flash) by 7.33 points.

### 1. 核心问题与整体含义
*   **研究动机**：现有的大语言模型（LLM）记忆智能体大多是“被动反应式”的。它们仅从对话历史中提取信息，记忆增长受限于偶然出现的输入。当遇到不确定性或推理失败时，这些智能体不会主动寻求外部知识。
*   **核心问题**：如何构建一个能够**主动获取、验证并高效管理**知识的自主记忆智能体，使其在不更新模型参数的情况下，通过外部存储实现持续进化？
*   **整体含义**：本文提出了 **U-Mem** 框架，标志着记忆智能体从“被动记录”向“主动学习”的范式转变。

### 2. 方法论
U-Mem 的核心由三个关键机制组成：
*   **成本感知的知识提取级联（Cost-Aware Cascade）**：
    *   当智能体推理失败时，它不会盲目自省，而是按成本从低到高触发级联：**Level 1 (教师 LLM)** -> **Level 2 (工具增强型教师，如代码解释器)** -> **Level 3 (人类专家/最强模型模拟)**。
    *   通过对比失败轨迹与获取的正确参考轨迹，提取“全局程序化记忆”（正确流程）和“局部纠错记忆”（针对特定错误的修正）。
*   **语义感知汤普森采样（SA-CTS）**：
    *   **解决冷启动**：传统检索基于相似度或固定效用值，导致新记忆因缺乏反馈而难以被选中。SA-CTS 将记忆效用建模为高斯分布 $N(\mu, \sigma^2)$，利用方差（不确定性）给予新记忆探索机会。
    *   **检索公式**：$Score(m_i) = (1 - \lambda) \cdot \text{sim}(q_t, m_i) + \lambda \cdot \tilde{u}_i$，其中 $\tilde{u}_i$ 是从后验分布中采样的效用值。
*   **流式记忆演化与维护**：
    *   包含**追加（Append）**、**合并（Merge，提升通用性）**和**剪枝（Prune，删除冗余）**操作，确保记忆库紧凑且信息密度高。

### 6. 主要结论与发现
*   **性能卓越**：U-Mem 在所有基准上均显著优于现有记忆方法。例如，在 HotpotQA 上将 Qwen2.5-7B 提升了 14.6 分。
*   **媲美 RL**：在不改变参数的情况下，U-Mem 的效果达到甚至超过了经过 GRPO 强化学习训练的模型。
*   **成本效益**：通过级联机制，U-Mem 仅需调用 23% 的昂贵专家模型，就能达到 99% 的全专家调用性能。
*   **失败是金**：实验证明，从失败中学习（对比学习）比仅从成功经验中学习对能力边界的扩展作用更大。

### 7. 优点
*   **主动性突破**：打破了记忆智能体被动接收信息的局限，引入了主动寻求外部验证的机制。
*   **解决冷启动**：引入汤普森采样解决了记忆检索中的“富者愈富”现象，使新知识能被有效利用。
*   **即插即用**：作为非参数化方案，它能直接应用于 DeepSeek 或 Gemini 等闭源模型，具有极强的工程实用价值。

### 8. 不足与局限
*   **专家依赖**：Level 3 级联依赖于更高水平的信号（如更强的模型或人类），如果外部信号本身存在偏差，记忆质量会受损。
*   **推理延迟**：虽然节省了训练成本，但检索、采样以及更长的上下文（包含记忆）会增加单次推理的延迟和 Token 消耗。
*   **任务相似度敏感**：实验显示，记忆带来的增益与测试集和训练流的语义相似度高度正相关，对于完全陌生的分布，增益可能受限。
