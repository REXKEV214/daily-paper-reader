## 摘要
生成式推荐（GenRec）模型通常通过全注意力机制对用户行为进行建模，但由于极高的计算成本和随机交互带来的噪声积累，难以扩展到终身序列。为了应对这些挑战，我们提出了 Rec2PM，这是一个将长用户交互历史压缩为紧凑偏好记忆（Preference Memory）标记的框架。与受限于串行训练的传统循环方法不同，Rec2PM 采用了一种新颖的自引用教师强制（self-referential teacher-forcing）策略：它利用历史的全局视图生成参考记忆，并将其作为并行化循环更新的监督目标。这使得模型在保持推理过程中迭代更新能力的同时，能够实现完全并行的训练。此外，通过将记忆表示为标记嵌入（token embeddings）而非庞大的 KV 缓存，Rec2PM 实现了极高的存储效率。在大规模基准测试上的实验表明，与全序列模型相比，Rec2PM 在显著降低推理延迟和内存占用的同时，实现了更优的准确率。分析表明，偏好记忆起到了去噪信息瓶颈（Information Bottleneck）的作用，有效地过滤了交互噪声，从而捕捉到稳健的长期兴趣。

## Abstract
Generative recommendation (GenRec) models typically model user behavior via full attention, but scaling to lifelong sequences is hindered by prohibitive computational costs and noise accumulation from stochastic interactions. To address these challenges, we introduce Rec2PM, a framework that compresses long user interaction histories into compact Preference Memory tokens. Unlike traditional recurrent methods that suffer from serial training, Rec2PM employs a novel self-referential teacher-forcing strategy: it leverages a global view of the history to generate reference memories, which serve as supervision targets for parallelized recurrent updates. This allows for fully parallel training while maintaining the capability for iterative updates during inference. Additionally, by representing memory as token embeddings rather than extensive KV caches, Rec2PM achieves extreme storage efficiency. Experiments on large-scale benchmarks show that Rec2PM significantly reduces inference latency and memory footprint while achieving superior accuracy compared to full-sequence models. Analysis reveals that the Preference Memory functions as a denoising Information Bottleneck, effectively filtering interaction noise to capture robust long-term interests.

### 1. 核心问题与整体含义（研究动机和背景）
生成式推荐（GenRec）目前面临处理“终身用户行为序列”的瓶颈。核心问题有二：
*   **计算与存储开销**：标准 Transformer 的全注意力机制具有平方级复杂度（$O(L^2)$），在处理数千甚至上万个交互标记时，推理延迟和 KV 缓存的存储压力在工业级场景下不可接受。
*   **噪声干扰**：用户行为具有随机性（如误点），直接对原始长历史进行全量注意力计算会放大噪声，干扰模型对用户核心偏好的捕捉。

**研究动机**：开发一种既能压缩长历史以降低成本，又能通过“信息瓶颈”效应过滤噪声，且支持高效并行训练的记忆增强型推荐框架。

### 2. 方法论：Rec2PM 框架
Rec2PM 的核心思想是将长历史压缩为少量的**偏好记忆标记（Preference Memory Tokens）**，并引入一种新颖的训练策略。

*   **三位一体记忆结构**：
    1.  **工作记忆（Working Memory）**：近期原始交互序列，保留高保真细节。
    2.  **偏好记忆（Preference Memory）**：从归档历史中提取的紧凑 Token 嵌入（如仅 4-20 个 Token）。
    3.  **参数记忆（Parametric Memory）**：模型共享的静态权重。
*   **自参考教师强制策略（Self-referential Teacher-forcing）**：这是解决循环记忆模型“训练难”的关键。
    *   **第一步：全局参考生成**。模型在训练时通过全量历史生成“参考记忆”（Reference Memory），这被视为低误差的“教师信号”。
    *   **第二步：并行更新优化**。将序列切段，模拟推理时的增量更新过程。要求模型基于“前一时刻的参考记忆”和“当前段数据”生成的“更新记忆”，去对齐（MSE Loss）第一步中的“参考记忆”。
*   **更新机制**：支持**覆盖（Overwriting）**和**追加（Appending）**两种模式。实验证明覆盖模式（保持固定数量的 Token）在去噪和效率上表现更优。
