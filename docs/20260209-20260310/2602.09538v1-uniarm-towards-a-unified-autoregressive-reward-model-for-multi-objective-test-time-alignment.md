## 摘要
多目标对齐旨在使大语言模型（LLM）的响应与多个人类偏好目标保持一致。在现有方法中，通过自回归奖励模型（ARM）引导冻结的 LLM 生成以实现多目标测试时对齐，是一种低成本的解决方案。然而，这些方法通常为每个偏好目标依赖独立的参数，要么在不同偏好维度上独立训练 ARM（这忽略了偏好特征之间的交互），要么为每个偏好训练带有独立特征提取模块的单个 ARM（这可能导致特征纠缠）。这两种策略都可能导致生成输出与用户偏好之间出现失配。为了解决这一局限性，我们提出了用于 ARM 训练的偏好调制与共享低秩自适应（MoSLoRA），该方法首先通过偏好无关模块提取共享特征，然后通过以混合偏好向量为条件的偏好调制模块对共享特征应用仿射变换。这种设计减轻了特征纠缠，并能够在推理过程中精确控制偏好权衡。在此基础上，我们引入了统一自回归奖励模型（UniARM），这是一个用于多目标测试时对齐的新型框架。UniARM 在单个参数空间中对所有偏好维度进行联合建模，消除了为每个偏好目标设置独立参数的需求。在更大规模的 LLM 上进行的实验增强了其实用性。

## Abstract
Multi-objective alignment aims to align LLM responses with multiple human preference objectives. Among existing methods, guiding the generation of frozen LLMs through autoregressive reward models (ARMs) to accomplish multi-objective test-time alignment is a low-cost solution. However, these methods typically rely on independent parameters for each preference objective, either by training ARMs independently across preference dimensions, which neglects interactions among preference features, or by training a single ARM with separate feature extraction modules for each preference, which can cause feature entanglement. Both strategies can result in misalignment between generated outputs and user preferences. To address this limitation, we propose Preference-Modulated \& Shared Low-Rank Adaptation (MoSLoRA) for ARM training, which first extracts shared features via a preference-agnostic module and then applies affine transformations to shared features via a preference modulation module conditioned on mixed preference vectors. This design mitigates feature entanglement and enables precise control over preference trade-offs during inference. Building on this, we introduce the Unified Autoregressive Reward Model (UniARM), a novel framework for multi-objective test-time alignment. UniARM jointly models all preference dimensions in a single parameter space, eliminating the need for independent parameters for each preference objective. es on larger-scale LLMs, enhancing its practical usability.

论文关注的是大语言模型（LLM）的**多目标对齐**问题，即如何让模型同时满足多个（有时是冲突的）人类偏好，如“有用性”与“无害性”。
*   **研究动机**：现有的测试时对齐方法（如 GenARM）通常为每个目标训练独立的自回归奖励模型（ARM），这会导致推理成本激增。改进方案（如 PARM）虽然尝试在单个模型中处理多目标，但其采用的独立特征提取模块容易导致**特征纠缠**（Feature Entanglement），即模型难以准确区分和权衡不同的偏好维度，从而限制了对齐的精度。
*   **核心目标**：开发一种统一的奖励模型，在单一参数空间内实现多偏好维度的联合建模，同时消除特征纠缠，提高推理效率和对齐效果。

### 2. 方法论：核心思想与关键技术
论文提出了 **UniARM** 框架，其核心技术是**偏好调制共享低秩自适应（MoSLoRA）**：
*   **核心思想**：不再为每个偏好分配独立的参数子空间，而是通过“共享特征提取 + 动态特征调制”的方式处理多目标。
*   **关键技术细节**：
    *   **共享模块（Preference-Agnostic Module）**：使用标准的 LoRA 结构提取所有偏好共有的语义特征 $h'$。
    *   **调制模块（Preference-Modulation Module）**：根据输入的混合偏好向量 $o'$（由各目标描述的嵌入加权得到），生成调制参数 $\gamma$（缩放）和 $\eta$（偏移）。
    *   **仿射变换公式**：最终特征 $\tilde{h} = (\gamma_{o'} + 1) \odot h' + \eta_{o'}$。这种设计允许模型在同一特征空间内，通过仿射变换精确地调整不同偏好的权重。
    *   **训练策略**：采用联合损失函数，结合了针对单个维度的“局部损失”和针对整体偏好权衡的“全局损失”（作为正则化项），以确保 Pareto 前沿的平滑性。
    *   **推理合并**：在推理时，共享模块可合并至原模型权重，调制模块通过矩阵变换减少额外延迟。

### 6. 主要结论与发现
*   **性能提升**：在安全对齐任务中，UniARM 相比 PARM 在 HV 和 MIP 指标上分别提升了 **18.5%** 和 **30.2%**。
*   **弱引导强效果显著**：小规模 UniARM 能有效提升大规模冻结 LLM 的对齐表现，HV 提升达 9.1%。
*   **解决特征纠缠**：定性分析显示，UniARM 能够根据偏好向量的变化产生平滑且单调的响应变化，而 PARM 则会出现非单调的波动，证明了 UniARM 调制机制的有效性。
*   **零额外开销**：在不增加推理延迟和可训练参数量的前提下，实现了更优的对齐效果。

### 7. 优点（亮点）
*   **架构创新**：将多任务学习中的“特征调制”思想引入奖励模型，通过仿射变换替代了简单的参数线性组合，有效解决了特征纠缠问题。
*   **参数高效**：在单一参数空间内处理无限种偏好组合，存储和计算成本极低。
*   **实用性强**：支持“弱引导强”模式，这意味着开发者可以用较小的代价优化超大规模模型的测试时表现。

### 8. 不足与局限
*   **任务覆盖面**：目前的实验主要集中在人类偏好对齐（安全、有用、幽默），尚未在更复杂的推理任务或长文本生成任务中验证其权衡能力。
*   **偏好描述依赖**：模型依赖于预定义的偏好文本描述（Objective Descriptions）来生成嵌入，对于模糊或高度重叠的偏好定义，其调制效果可能会受限。
*   **测试时延迟**：虽然比多模型方法快，但相比于直接微调好的策略模型，测试时引导（Guided Decoding）仍然会带来额外的计算开销。
