## 摘要
虽然多模态大语言模型在短期推理方面表现出色，但由于上下文窗口有限以及无法反映人类认知效率的静态记忆机制，它们在长程视频理解方面仍面临挑战。现有的范式通常处于两个极端：以视觉为中心的方法通过密集的视觉累积导致高延迟和冗余；而以文本为中心的方法则通过激进的字幕生成导致细节丢失和幻觉。为了弥补这一差距，我们提出了 MM-Mem，这是一种基于模糊痕迹理论（Fuzzy-Trace Theory）的金字塔型多模态记忆架构。MM-Mem 将记忆分层构建为感官缓冲（Sensory Buffer）、情节流（Episodic Stream）和符号图式（Symbolic Schema），实现了将细粒度感知痕迹（verbatim）逐步蒸馏为高层语义图式（gist）。此外，为了管理记忆的动态构建，我们推导出了一个语义信息瓶颈目标，并引入了 SIB-GRPO，以优化记忆压缩与任务相关信息保留之间的权衡。在推理过程中，我们设计了一种熵驱动的自顶向下记忆检索策略，该策略首先尝试抽象的符号图式，并在不确定性较高时逐步“向下钻取”至感官缓冲和情节流。在 4 个基准测试上的广泛实验证实了 MM-Mem 在离线和流式任务中的有效性，展示了强大的泛化能力，并验证了受认知启发的记忆组织的有效性。

## Abstract
While multimodal large language models have demonstrated impressive short-term reasoning, they struggle with long-horizon video understanding due to limited context windows and static memory mechanisms that fail to mirror human cognitive efficiency. Existing paradigms typically fall into two extremes: vision-centric methods that incur high latency and redundancy through dense visual accumulation, or text-centric approaches that suffer from detail loss and hallucination via aggressive captioning. To bridge this gap, we propose MM-Mem, a pyramidal multimodal memory architecture grounded in Fuzzy-Trace Theory. MM-Mem structures memory hierarchically into a Sensory Buffer, Episodic Stream, and Symbolic Schema, enabling the progressive distillation of fine-grained perceptual traces (verbatim) into high-level semantic schemas (gist). Furthermore, to govern the dynamic construction of memory, we derive a Semantic Information Bottleneck objective and introduce SIB-GRPO to optimize the trade-off between memory compression and task-relevant information retention. In inference, we design an entropy-driven top-down memory retrieval strategy, which first tries with the abstract Symbolic Schema and progressively "drills down" to the Sensory Buffer and Episodic Stream under high uncertainty. Extensive experiments across 4 benchmarks confirm the effectiveness of MM-Mem on both offline and streaming tasks, demonstrating robust generalization and validating the effectiveness of cognition-inspired memory organization. Code is available at https://github.com/EliSpectre/MM-Mem.

## 1. 核心问题与整体含义
论文针对多模态大语言模型（MLLM）在**长程视频理解（Long-Horizon Video Understanding）**中的局限性提出解决方案。核心问题在于：现有的记忆机制要么过于依赖视觉（导致冗余和高延迟），要么过于依赖文本（导致细节丢失和幻觉）。

研究动机源于认知心理学中的**模糊痕迹理论（Fuzzy-Trace Theory）**，该理论认为人类记忆由“表层细节（Verbatim）”和“语义要义（Gist）”并行组成。论文旨在构建一个能够像人类一样在保留关键视觉证据的同时，高效提取高层语义信息的智能体记忆系统。

## 2. 方法论
论文提出了 **MM-Mem** 架构，其核心思想是构建一个自底向上的金字塔型记忆体系，并配合自顶向下的检索策略。

### 核心架构：三层金字塔
*   **感官缓冲（Sensory Buffer, L1）：** 存储细粒度的视觉证据（关键帧/子片段）及其关联的原始文本（如字幕）。
*   **情节流（Episodic Stream, L2）：** 通过对感官数据进行事件级整合，形成紧凑的事件序列和摘要。
*   **符号图式（Symbolic Schema, L3）：** 构建基于实体的知识图谱，实现跨情节的高层语义抽象。

### 关键技术：SIB-GRPO
*   **语义信息瓶颈（Semantic Information Bottleneck, SIB）：** 基于信息瓶颈理论，推导出在限制记忆长度（压缩）的同时最大化保留任务相关语义（预测能力）的目标函数。
*   **SIB-GRPO 算法：** 引入“组相对策略优化（GRPO）”强化学习算法，通过奖励函数（包含任务准确度奖励、长度惩罚和 KL 散度约束）来训练记忆管理器，实现动态的记忆构建与压缩。

### 检索策略：熵驱动的自顶向下检索
*   智能体首先查询最抽象的 L3 层。
*   计算预测结果的**熵（Entropy）**，若不确定性较高（熵值超过阈值），则依次向下“钻取”至 L2 和 L1 层，获取更细粒度的视觉证据进行验证。

## 6. 主要结论与发现
*   **性能领先：** MM-Mem 在 Video-MME 和 MLVU 上显著优于现有的开源智能体系统，甚至在某些指标上接近闭源模型。
*   **记忆分层有效性：** 视觉记忆对长视频至关重要（移除后性能大幅下降），而 SIB-GRPO 显著提升了模型处理长程依赖的能力。
*   **检索效率：** 熵驱动的策略能有效平衡计算成本与推理精度，在需要时才调用底层视觉细节。
*   **领域差异：** 模型在自我中心视频（Ego）中表现极佳，但在电影场景（Movie）中由于剪辑和叙事逻辑复杂，提升幅度相对较小。

## 7. 优点
*   **理论支撑扎实：** 结合了认知科学（模糊痕迹理论）与信息论（信息瓶颈），方法论具有很强的解释性。
*   **动态管理：** 相比静态记忆，SIB-GRPO 实现了记忆的动态蒸馏，减少了冗余。
*   **多粒度对齐：** 成功实现了视觉“表层”与文本“要义”的跨模态层级对齐。

## 8. 不足与局限
*   **构建开销：** 金字塔记忆的构建过程（尤其是 L3 的图谱提取）相比扁平化模型具有更高的初始计算成本。
*   **感知依赖：** 系统的性能高度依赖于上游视觉编码器和字幕生成器的质量。
*   **泛化限制：** 目前主要在受限任务（VQA）中通过强化学习对齐，在完全无监督或终身学习场景下的表现尚待验证。
*   **场景偏差：** 在处理具有复杂叙事结构的电影类视频时，现有的记忆设计仍有优化空间。
