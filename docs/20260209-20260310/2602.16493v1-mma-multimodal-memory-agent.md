## 摘要
长程多模态智能体依赖于外部记忆；然而，基于相似性的检索往往会提取出陈旧、低可信度或冲突的项，这可能引发过度自信的错误。我们提出了多模态记忆智能体（MMA），它通过结合来源可信度、时间衰减和冲突感知的网络共识，为每个检索到的记忆项分配动态可靠性评分，并利用该信号重新加权证据，在支持不足时选择弃权。我们还引入了 MMA-Bench，这是一个通过程序生成的基准测试，用于评估具有受控说话者可靠性和结构化文本-视觉矛盾的信念动力学。利用该框架，我们揭示了“视觉安慰剂效应”（Visual Placebo Effect），展示了基于检索增强生成（RAG）的智能体如何从基础模型中继承潜在的视觉偏见。在 FEVER 数据集上，MMA 在保持与基线相当的准确率的同时，将方差降低了 35.2% 并提高了选择性效用；在 LoCoMo 上，面向安全的配置提高了可执行准确率并减少了错误回答；在 MMA-Bench 上，MMA 在视觉模式下达到了 41.18% 的 Type-B 准确率，而基线在相同协议下则崩溃至 0.0%。代码：https://github.com/AIGeeksGroup/MMA。

## Abstract
Long-horizon multimodal agents depend on external memory; however, similarity-based retrieval often surfaces stale, low-credibility, or conflicting items, which can trigger overconfident errors. We propose Multimodal Memory Agent (MMA), which assigns each retrieved memory item a dynamic reliability score by combining source credibility, temporal decay, and conflict-aware network consensus, and uses this signal to reweight evidence and abstain when support is insufficient. We also introduce MMA-Bench, a programmatically generated benchmark for belief dynamics with controlled speaker reliability and structured text-vision contradictions. Using this framework, we uncover the "Visual Placebo Effect", revealing how RAG-based agents inherit latent visual biases from foundation models. On FEVER, MMA matches baseline accuracy while reducing variance by 35.2% and improving selective utility; on LoCoMo, a safety-oriented configuration improves actionable accuracy and reduces wrong answers; on MMA-Bench, MMA reaches 41.18% Type-B accuracy in Vision mode, while the baseline collapses to 0.0% under the same protocol. 长程多模态智能体（Multimodal Agents）在处理复杂任务时高度依赖外部记忆系统。然而，现有的记忆增强架构（如基于 RAG 的系统）存在一个核心缺陷：**盲目信任检索结果**。
*   **研究动机**：传统的基于相似度的检索往往会召回陈旧、低信度或相互冲突的信息。智能体如果将这些信息视为同等可靠，就会在推理中放大错误，产生“过度自信的幻觉”。
*   **核心挑战**：如何在嘈杂、冲突的多模态输入中识别可靠证据，并在证据不足时让智能体学会“明智地弃权”。

## 2. 方法论：核心思想与关键技术
MMA 提出了一种**推理时置信度评分框架**，在记忆项级别进行可靠性评估。

### 核心思想
MMA 不再简单地根据向量相似度注入记忆，而是为每个检索到的记忆项 $M_i$ 计算一个动态可靠性分数 $C(M_i)$，用于重新加权证据并调节弃权行为。

### 关键技术细节
可靠性分数由三个维度加权合成：
1.  **来源可靠性 $S(M_i)$**：根据记忆来源（如不同用户或渠道）预设的信誉先验进行映射。
2.  **时间衰减 $T(M_i)$**：采用指数衰减函数 $T(M_i) = \exp(-\frac{\ln(2)}{T_{half}} \Delta t_i)$，模拟信息随时间流逝而过时的特性。
3.  **网络共识 $C_{con}(M_i)$**：通过计算检索邻域内的语义一致性来过滤冲突。公式考虑了邻居项的置信度与余弦相似度，正值强化信心，负值（冲突）则惩罚置信度。

### 决策流程
智能体根据综合得分对记忆进行重排或过滤。如果所有证据的置信度均低于阈值，智能体将触发弃权机制，输出“信息不足”而非猜测。

*   **精细化建模**：将记忆可靠性拆解为来源、时间和共识三个维度，符合人类处理信息的逻辑。
