## 摘要
长程多模态智能体依赖于外部记忆；然而，基于相似性的检索往往会提取出陈旧、低可信度或冲突的项，这可能引发过度自信的错误。我们提出了多模态记忆智能体（MMA），它通过结合来源可信度、时间衰减和冲突感知的网络共识，为每个检索到的记忆项分配动态可靠性评分，并利用该信号重新加权证据，在支持不足时选择弃权。我们还引入了 MMA-Bench，这是一个通过程序生成的基准测试，用于评估具有受控说话者可靠性和结构化文本-视觉矛盾的信念动力学。利用该框架，我们揭示了“视觉安慰剂效应”（Visual Placebo Effect），展示了基于检索增强生成（RAG）的智能体如何从基础模型中继承潜在的视觉偏见。在 FEVER 数据集上，MMA 在保持与基线相当的准确率的同时，将方差降低了 35.2% 并提高了选择性效用；在 LoCoMo 上，面向安全的配置提高了可执行准确率并减少了错误回答；在 MMA-Bench 上，MMA 在视觉模式下达到了 41.18% 的 Type-B 准确率，而基线在相同协议下则崩溃至 0.0%。代码：https://github.com/AIGeeksGroup/MMA。

## Abstract
Long-horizon multimodal agents depend on external memory; however, similarity-based retrieval often surfaces stale, low-credibility, or conflicting items, which can trigger overconfident errors. We propose Multimodal Memory Agent (MMA), which assigns each retrieved memory item a dynamic reliability score by combining source credibility, temporal decay, and conflict-aware network consensus, and uses this signal to reweight evidence and abstain when support is insufficient. We also introduce MMA-Bench, a programmatically generated benchmark for belief dynamics with controlled speaker reliability and structured text-vision contradictions. Using this framework, we uncover the "Visual Placebo Effect", revealing how RAG-based agents inherit latent visual biases from foundation models. On FEVER, MMA matches baseline accuracy while reducing variance by 35.2% and improving selective utility; on LoCoMo, a safety-oriented configuration improves actionable accuracy and reduces wrong answers; on MMA-Bench, MMA reaches 41.18% Type-B accuracy in Vision mode, while the baseline collapses to 0.0% under the same protocol. ### 1. 核心问题与整体含义
在长程交互的多模态智能体中，依赖外部记忆进行检索增强生成（RAG）已成为主流。然而，现有的记忆系统存在一个核心缺陷：**盲目信任检索结果**。基于相似度的检索往往会引入过时、低可信度或相互冲突的信息，导致智能体在推理时产生“过度自信的错误”（Over-confident errors）。
论文的研究动机在于：
*   **记忆可靠性缺失**：现有系统将所有检索到的记忆项视为同等可靠。
*   **认知失调**：当文本记忆与视觉证据冲突时，智能体往往无法有效仲裁。
*   **缺乏弃权机制**：智能体在证据不足时仍倾向于给出确定的错误答案。

### 2. 方法论：核心思想与技术细节
MMA 框架在传统的记忆检索层之上增加了一个**元认知可靠性层**，通过动态评分来调节推理过程。

*   **核心思想**：为每个检索到的记忆项 $M_i$ 计算一个动态可靠性评分 $C(M_i) \in [0, 1]$，用于重新加权证据，并在置信度过低时触发弃权。
*   **关键技术细节（评分公式）**：
    可靠性评分由三个维度加权组成：$C(M_i) = w'_s S(M_i) + w'_t T(M_i) + w'_c C_{con}(M_i)$
    1.  **来源可靠性 $S(M_i)$**：根据记忆来源（如不同用户或数据库）预设的信任先验。
    2.  **时间衰减 $T(M_i)$**：采用指数衰减模型 $exp(-\frac{ln(2)}{T_{half}} \Delta t_i)$，模拟信息随时间失效的过程。
    3.  **网络共识 $C_{con}(M_i)$**：衡量该记忆项在检索到的语义邻域内获得的支持度。通过余弦相似度计算支持因子，正值强化置信度，负值（冲突）则惩罚置信度。
*   **决策逻辑**：高置信度项被优先用于推理；低置信度或冲突严重的项会导致智能体选择“不知道”或“信息不足”（弃权）。

### 6. 主要结论与发现
*   **稳定性提升**：在 FEVER 上，MMA 在保持准确率的同时，将性能方差降低了 **35.2%**。
*   **视觉安慰剂效应（Visual Placebo Effect）**：研究发现，仅仅是视觉数据的存在（即使是模糊或无关的），就会诱导 RAG 智能体产生虚假的确定性。MMA 通过共识机制有效缓解了这一偏见。
*   **冲突处理能力**：在 MMA-Bench 的“可靠性反转”测试中，基线模型因无法处理冲突而崩溃（准确率 0%），而 MMA 达到了 **41.18%** 的准确率。
*   **安全性**：在 LoCoMo 任务中，MMA 的安全配置显著减少了错误回答的数量，提升了可执行准确率。

### 7. 优点
*   **认知深度**：不只是做检索，而是引入了类似人类的“元认知”机制，对记忆质量进行多维度评估。
*   **基准创新**：MMA-Bench 填补了多模态长程记忆冲突评估的空白，能够精准诊断智能体的“盲目自信”。
*   **即插即用**：作为推理时模块，可以增强现有的 RAG 架构，而无需重新训练基础大模型。

### 8. 不足与局限
*   **检索依赖性**：MMA 是一个后检索（Post-retrieval）模块，如果底层的检索系统完全没能召回相关证据，MMA 无法凭空修复缺失的信息。
*   **稀疏性权衡**：在信息非常稀疏的环境下，严格的“网络共识”机制可能导致智能体过于保守，从而漏掉一些虽然孤立但正确的证据。
*   **先验依赖**：来源可靠性评分目前仍依赖预设或静态映射，未来需要更动态的来源信用评估机制。
