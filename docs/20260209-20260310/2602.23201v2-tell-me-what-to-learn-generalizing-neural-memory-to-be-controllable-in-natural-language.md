## 摘要
现代机器学习模型被部署在多样化且非平稳的环境中，必须不断适应新任务和不断演进的知识。持续微调和上下文学习成本高昂且脆弱，而神经记忆方法则有望实现轻量级更新并最大限度地减少遗忘。然而，现有的神经记忆模型通常假设单一的固定目标和同质信息流，导致用户无法控制模型随时间推移记住或忽略的内容。为了应对这一挑战，我们提出了一种通用的神经记忆系统，该系统能够根据自然语言指定的学习指令执行灵活的更新。我们的方法使自适应智能体能够从异构信息源中进行选择性学习，从而支持医疗保健和客户服务等固定目标记忆更新不足的应用场景。

## Abstract
Modern machine learning models are deployed in diverse, non-stationary environments where they must continually adapt to new tasks and evolving knowledge. Continual fine-tuning and in-context learning are costly and brittle, whereas neural memory methods promise lightweight updates with minimal forgetting. However, existing neural memory models typically assume a single fixed objective and homogeneous information streams, leaving users with no control over what the model remembers or ignores over time. To address this challenge, we propose a generalized neural memory system that performs flexible updates based on learning instructions specified in natural language. Our approach enables adaptive agents to learn selectively from heterogeneous information sources, supporting settings, such as healthcare and customer service, where fixed-objective memory updates are insufficient.

*   **研究动机**：现代 LLM 部署在动态环境中，需要不断吸收新知识。现有的适配方法存在明显缺陷：**微调（Fine-tuning）**成本高且易导致灾难性遗忘；**上下文学习（ICL）**受限于窗口长度且计算开销随文档增加呈平方级增长；**检索增强生成（RAG）**依赖检索质量且难以处理风格或行为层面的适配。
*   **核心问题**：现有的**神经记忆（Neural Memory）**方法虽然高效，但通常采用固定的更新目标（如预测下一个 Token），用户无法干预模型记忆的内容。例如，在医疗场景下，用户可能希望模型学习“何时转诊”的逻辑，但忽略文档中“过时的用药剂量”。
*   **整体含义**：本文旨在将神经记忆泛化为一种**自然语言可控**的系统，让用户通过指令（Instruction）精确引导模型对每一条新信息的存储、忽略或抑制。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：将记忆更新过程建模为一个由指令驱动的参数化函数。
*   **数学形式**：定义更新规则 $U_\psi$ 为：$M_t = U_\psi(M_{t-1}, I_t, D_t)$。其中 $M$ 是记忆状态，$I$ 是自然语言指令，$D$ 是新文档。
*   **关键技术细节**：
    *   **架构基础**：基于 **MemoryLLM**（以 Llama-3 为骨干），将神经记忆实现为预置在 Transformer 每一层中的可学习嵌入（Embedding）。
    *   **指令调节**：在“学习阶段”，模型同时输入指令和文档。指令通过注意力机制调制文档信息的压缩过程，决定哪些信息进入 256 个记忆 Token 槽位。
    *   **训练目标**：使用**序列损失函数（Sequence Loss）**，通过探测查询（Probes）来优化。探测包括：
        1.  **正向查询**：验证是否记住了指令要求学习的事实。
        2.  **负向/控制查询**：验证是否成功忽略了指令要求忽略的信息，或是否保持了原始知识（特异性）。
    *   **两阶段训练**：模型首先在指令下进行记忆写入，随后在推理阶段利用更新后的记忆回答问题，梯度回传同时优化写入逻辑和推理逻辑。

### 3. 实验设计
*   **数据集/场景**：构建了一个基于 **CounterFACT** 的合成基准测试。
    *   包含 21,918 个事实，分为 16 个语义类别（如地理、职业、语言等）。
    *   **OOD（分布外）测试**：预留了 4 个类别和相关指令完全不参与训练，专门用于测试模型对**未见指令**的泛化能力。
*   **实验任务**：
    1.  **事实学习**：选择性记忆特定类别的事实。
    2.  **风格与行为学习**：学习文档的 Markdown 格式或特定的拒绝回答（Refusal）行为。
    3.  **组合泛化**：同时处理多个指令（如“学习 A 类事实，但拒绝回答 B 类问题”）。
*   **对比方法（Baselines）**：
    *   **MemoryLLM**：原始的固定目标神经记忆。
    *   **ICL-FT**：经过微调的上下文学习（将所有历史文档和指令放入 Context）。
    *   **RAG-FT**：经过微调的检索增强生成。

### 4. 资源与算力
*   **硬件**：使用了 **6 台 NVIDIA A100 GPU**。
*   **训练细节**：
    *   采用 **8-bit AdamW** 优化器以节省显存。
    *   有效 Batch Size 为 6 个序列（共 24 篇文档）。
    *   使用了梯度累积（Gradient Accumulation）和早停（Early Stopping）策略。
    *   所有模型在 10 个 Epoch 内收敛。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了多维度评估，包括 10 步和 20 步的持续学习序列测试。
*   **充分性分析**：
    *   **消融实验**：对比了“仅优化推理步”与“同时优化记忆写入步”的效果，证明了学习“如何写入”是关键。
    *   **层级分析**：对 Transformer 的 32 个层进行了因果干预实验，分析了指令在不同层是如何被处理的。
    *   **公平性**：为了公平对比，研究者使用 GPT-5.1 为 ICL 和 RAG 基准优化了 5 种不同的 Prompt，并选取表现最好的一种进行微调。
    *   **结论**：实验设计严谨，通过 OOD 类别和合成干扰项有效排除了模型通过“捷径”获胜的可能性。

### 6. 主要结论与发现
*   **泛化能力强**：GNM 能够完美泛化到训练中从未见过的自然语言指令和事实类别。
*   **选择性优异**：在“忽略无关信息”的能力上，GNM 显著优于 ICL 和 RAG。ICL 即使经过微调，也容易受到上下文中无关事实的干扰。
*   **效率优势**：GNM 的推理成本不随记忆文档数量增加而增加（$O(1)$ 复杂度），而 ICL 随文档增加计算量剧增。
*   **内部机制**：发现模型存在“两阶段”处理模式：**第 5-14 层**主要负责解析和表示学习指令，**第 15-31 层**则利用指令表示来选择性地将文档信息写入记忆。

### 7. 优点
*   **创新性**：首次将自然语言指令引入神经记忆的更新控制，填补了该领域的空白。
*   **高效率与高精度并存**：在保持神经记忆低推理成本的同时，达到了接近甚至超过全量上下文学习的准确度。
*   **可解释性探索**：通过层级消融和向量对齐分析，深入探讨了指令如何干预记忆形成的物理过程。

### 8. 不足与局限
*   **合成数据依赖**：实验主要在合成的 CounterFACT 变体上进行，尚未在真实世界的大规模异构数据集（如医疗病历库）上完全验证。
*   **记忆容量限制**：由于采用固定数量的记忆 Token，当学习序列极长（如超过 20 步）时，旧记忆会被新记忆覆盖，导致性能下降。
*   **时序矛盾处理**：目前模型难以处理存在逻辑冲突或需要严格时序更新的信息（例如：先说 A 是对的，后说 A 是错的）。
*   **应用限制**：目前仅基于 Llama-3 架构，对于其他非 Transformer 架构或超大规模模型的适用性有待观察。
