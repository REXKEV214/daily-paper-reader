## 摘要
大语言模型（LLMs）的最新进展通过丰富的语义理解和上下文推理，捕捉用户兴趣和物品特征的细微语义，为推荐系统提供了新的机遇。特别是，LLMs 已被用作重排序器（rerankers），根据推断的用户-物品相关性对候选物品进行重新排序。然而，这些方法通常需要昂贵的在线推理阶段推理，导致高延迟，阻碍了实际部署。在这项工作中，我们提出了 Persona4Rec，这是一个执行离线推理以构建物品可解释画像（persona）表示的推荐框架，从而实现轻量级且可扩展的实时推理。在离线阶段，Persona4Rec 利用 LLMs 对物品评论进行推理，推断出解释不同类型用户为何与物品互动的多样化用户动机；这些推断出的动机被具体化为画像表示，为每个物品提供多个可供人类解释的视角。与依赖单一物品表示的传统方法不同，Persona4Rec 学习通过专用编码器将用户画像（user profiles）与最合理的物品端画像进行对齐，有效地将用户-物品相关性转化为用户-画像相关性。在在线阶段，这种画像化的物品索引允许在不调用昂贵的 LLM 推理的情况下进行快速的相关性计算。大量实验表明，Persona4Rec 在大幅减少推理时间的同时，实现了与近期基于 LLM 的重排序器相当的性能。此外，定性分析证实，画像表示不仅驱动了高效的评分，还提供了直观的、基于评论的解释。这些结果表明，Persona4Rec 为下一代推荐系统提供了一种实用且可解释的解决方案。

## Abstract
Recent advances in large language models (LLMs) offer new opportunities for recommender systems by capturing the nuanced semantics of user interests and item characteristics through rich semantic understanding and contextual reasoning. In particular, LLMs have been employed as rerankers that reorder candidate items based on inferred user-item relevance. However, these approaches often require expensive online inference-time reasoning, leading to high latency that hampers real-world deployment. In this work, we introduce Persona4Rec, a recommendation framework that performs offline reasoning to construct interpretable persona representations of items, enabling lightweight and scalable real-time inference. In the offline stage, Persona4Rec leverages LLMs to reason over item reviews, inferring diverse user motivations that explain why different types of users may engage with an item; these inferred motivations are materialized as persona representations, providing multiple, human-interpretable views of each item. Unlike conventional approaches that rely on a single item representation, Persona4Rec learns to align user profiles with the most plausible item-side persona through a dedicated encoder, effectively transforming user-item relevance into user-persona relevance. At the online stage, this persona-profiled item index allows fast relevance computation without invoking expensive LLM reasoning. Extensive experiments show that Persona4Rec achieves performance comparable to recent LLM-based rerankers while substantially reducing inference time. Moreover, qualitative analysis confirms that persona representations not only drive efficient scoring but also provide intuitive, review-grounded explanations. These results demonstrate that Persona4Rec offers a practical and interpretable solution for next-generation recommender systems.

*   **研究动机**：虽然大语言模型（LLM）在推荐系统重排序（Reranking）中表现出卓越的语义理解和推理能力，但现有的 LLM 重排序方法通常依赖于**在线推理（Online Reasoning）**。这意味着在用户请求时，系统需要实时调用 LLM 处理复杂的文本输入（如评论、历史记录），导致极高的延迟和计算成本，难以在真实生产环境中大规模部署。
*   **核心问题**：如何在保留 LLM 深度语义推理能力的同时，消除在线阶段的推理负担，实现低延迟、高扩展性且具备可解释性的实时推荐？

### 2. 方法论：Persona4Rec 框架
Persona4Rec 的核心思想是将昂贵的 LLM 推理过程**移至离线阶段**，通过构建“画像化物品索引”来简化在线匹配。

*   **离线推理阶段（Offline Reasoning）**：
    1.  **画像构建（Persona Construction）**：利用 LLM 结合物品元数据（客观信息）和用户评论（主观信息），为每个物品生成 2-7 个不同的“用户画像（Personas）”。每个画像包含名称、描述和偏好理由，代表了不同类型用户喜欢该物品的潜在动机。
    2.  **用户-画像对齐（User-Persona Alignment）**：使用 LLM 作为评判者（LLM-as-a-judge），将用户的历史交互记录与目标物品的多个画像进行匹配，选出最能解释该用户行为的画像，生成监督信号。
    3.  **编码器训练（Encoder Training）**：基于上述对齐信号，使用对比学习（Contrastive Learning）训练一个轻量级双塔编码器（如 BGE-M3）。该编码器学习将用户特征和物品画像映射到统一的向量空间。
*   **在线推理阶段（Online Inference）**：
    1.  **轻量匹配**：系统不再调用 LLM，而是通过编码器快速生成当前用户的嵌入向量。
    2.  **高效评分**：计算用户向量与候选物品预存画像向量之间的最大余弦相似度。得分最高的画像不仅决定了排名，还直接作为推荐理由输出。

### 6. 主要结论与发现
*   **性能卓越**：Persona4Rec 在准确率上与最先进的在线 LLM 重排序器相当，甚至在某些指标上有所超越。
*   **极低延迟**：在线推理时间缩短至 **0.5ms - 0.75ms**，内存占用仅为 0.1GB，完全满足实时系统要求。
*   **冷启动友好**：在长尾物品（评论稀少）和冷启动场景（仅有元数据）下，通过语义总结仍能保持优于传统 CF 模型的表现。

### 7. 优点
*   **架构创新**：成功实现了“离线重推理，在线轻匹配”的解耦，解决了 LLM 落地推荐系统的最大痛点——延迟。
*   **可解释性强**：推荐理由直接源于预构建的、基于真实评论的画像，比单纯生成的文本更具事实依据。
*   **工业实用性**：对算力和内存的要求极低，且对缺失评论的场景有良好的鲁棒性。

### 8. 不足与局限
*   **画像数量上限**：为了计算效率，每个物品的画像数量被限制在 7 个以内。对于极其热门、受众极广的物品，这可能无法完全覆盖所有的用户动机。
*   **离线依赖**：虽然在线很快，但离线阶段构建画像和对齐仍需调用昂贵的 LLM API，对于拥有亿级物品库的平台，离线预计算成本依然存在。
*   **特异性权衡**：人工评估显示，由于画像是预构建的，其针对特定用户的“特异性（Specificity）”略低于实时生成的理由。
