## 摘要
具备价值意识的 AI 应当识别并适应不同用户的价值体系（基于价值的偏好）。这需要对价值进行操作化定义，而这往往容易出现规格错误。价值的社会属性要求其表示能够契合多位用户，尽管价值体系具有多样性，但在群体间仍表现出一定的模式。在序贯决策中，已有研究尝试通过不同智能体的演示来实现针对不同目标或价值的个性化。然而，这些方法要么需要手动设计特征，要么缺乏基于价值的可解释性和/或对多样化用户偏好的适应性。我们提出了基于聚类和基于偏好的多目标强化学习（PbMORL）的算法，用于在马尔可夫决策过程（MDPs）中学习智能体社会的价值对齐模型和价值体系。我们联合学习了社会衍生的价值对齐模型（groundings）以及一组能简洁代表社会中不同用户群体（聚类）的价值体系。每个聚类包含一个代表其成员基于价值的偏好的价值体系，以及一个反映与该价值体系相一致行为的近似帕累托最优策略。我们在两个具有人类价值的 MDPs 上，将我们的方法与最先进的 PbMORL 算法及基准方法进行了评估。

## Abstract
Value-aware AI should recognise human values and adapt to the value systems (value-based preferences) of different users. This requires operationalization of values, which can be prone to misspecification. The social nature of values demands their representation to adhere to multiple users while value systems are diverse, yet exhibit patterns among groups. In sequential decision making, efforts have been made towards personalization for different goals or values from demonstrations of diverse agents. However, these approaches demand manually designed features or lack value-based interpretability and/or adaptability to diverse user preferences.   We propose algorithms for learning models of value alignment and value systems for a society of agents in Markov Decision Processes (MDPs), based on clustering and preference-based multi-objective reinforcement learning (PbMORL). We jointly learn socially-derived value alignment models (groundings) and a set of value systems that concisely represent different groups of users (clusters) in a society. Each cluster consists of a value system representing the value-based preferences of its members and an approximately Pareto-optimal policy that reflects behaviours aligned with this value system. We evaluate our method against a state-of-the-art PbMORL algorithm and baselines on two MDPs with human values.

### 1. 核心问题与研究动机
*   **核心问题**：AI 如何在复杂的社会环境中识别并适应不同用户群体的多元价值体系（Value Systems）？
*   **研究背景**：
    *   **价值的多元性**：不同的人对“成就”、“仁慈”或“安全”等价值的优先级排序不同。
    *   **现有技术的局限**：目前的强化学习（RL）对齐方法通常假设全社会只有一套统一的奖励函数，或者需要手动设计特征，缺乏对群体价值模式的捕捉能力，且可解释性较差。
    *   **社会属性**：价值不仅是个人的，也是社会的。AI 需要理解价值的社会含义（Grounding），同时识别出具有相似偏好的不同群体（Clusters）。

### 2. 方法论：SVSL-P 算法
论文提出了 **SVSL-P（Social Value System Learning in MDPs）** 算法，其核心思想是将价值对齐视为一个**多目标强化学习（MORL）**问题，并通过聚类技术实现个性化。

*   **核心概念**：
    *   **社会基准（Social Grounding）**：用一个多目标奖励向量 $R(s, a)$ 表示不同行为与各项价值（如专业性、安全性）的对齐程度。
    *   **价值体系（Value System）**：通过线性加权向量 $W$ 来表示特定用户对各项价值的重视程度。
    *   **社会价值体系模型**：由一个社会基准、一组代表不同群体的权重向量 $\{W_l\}$ 以及将用户分配到群体的函数 $\beta$ 组成。
*   **关键技术流程**：
    1.  **初始化**：利用静态数据集进行初步的奖励向量和聚类估计。
    2.  **在线循环（HiL）**：
        *   **探索**：智能体在环境中执行策略并收集轨迹。
        *   **偏好收集**：向人类（或模拟代理）询问两段轨迹之间的偏好（基于特定价值的对齐偏好和基于整体价值体系的偏好）。
        *   **EM 算法更新**：**E步**将代理分配到最能代表其偏好的聚类；**M步**通过梯度下降更新奖励网络参数 $\theta$ 和聚类权重参数 $\omega$。
        *   **策略更新**：使用 **Envelope Q-Learning (EQL)** 学习一个参数化的多目标策略，使其能够根据不同的权重向量生成帕累托最优（Pareto-optimal）行为。

### 3. 实验设计
*   **实验场景**：

    *   **多值汽车环境（MVC）**：汽车需在“目标达成”、“安全性（避开行人）”和“舒适度（避开颠簸）”三个目标间权衡。
*   **Benchmark（基准方法）**：

    *   **PbMORL**：最先进的基于偏好的多目标强化学习算法（不含显式聚类）。
*   **对比指标**：代表性（Representativeness）、简洁性（Conciseness）、相干性（Coherence）、超体积（Hypervolume）和最大效用损失（MUL）。

### 6. 主要结论与发现
*   **SVSL-P 优于 SOTA**：在平衡“代表性”和“简洁性”的 Ray-Turi 指标上，SVSL-P 显著优于 PbMORL。
*   **高效的反馈机制**：SVSL-P 要求的反馈更符合人类直觉（仅询问用户自身的价值偏好），且在更少的查询下达到了相近或更好的帕累托前沿逼近效果。
*   **社会意识**：算法成功识别出了社会中潜在的价值群体模式，并为每个群体生成了定制化的最优策略。

### 7. 优点
*   **社会化建模**：首次在 PbMORL 框架中引入了显式的社会价值聚类，更符合人类社会的真实结构。
*   **高可解释性**：通过权重向量 $W$ 显式表达不同群体的偏好，使 AI 的决策逻辑透明化。
*   **减少人工负担**：在线 HiL 机制减少了对预定义奖励函数的依赖，且聚类机制减少了对每个个体进行独立学习的数据需求。

### 8. 不足与局限
*   **稳定性问题**：实验结果显示在不同随机种子下存在一定的波动，说明算法对初始化较为敏感。
*   **策略冗余**：有时学习到的两个不同价值体系（权重不同）在 MDP 中会映射到相同的物理策略，导致聚类效率下降。
*   **场景限制**：目前仅在合成的 MDP 环境中验证，尚未在具有高维状态空间的真实复杂场景（如 LLM 对齐）中大规模测试。
*   **线性假设**：假设价值体系是各项价值的线性加权，可能无法捕捉更复杂的非线性价值偏好。
