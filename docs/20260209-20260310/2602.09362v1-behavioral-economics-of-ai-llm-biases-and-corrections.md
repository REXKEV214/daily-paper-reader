## 摘要
生成式人工智能模型，特别是大语言模型（LLM），在经济和金融决策中是否表现出系统性的行为偏见？如果是这样，如何减轻这些偏见？借鉴认知心理学和实验经济学文献，我们针对主流 LLM 家族的不同版本和规模，进行了一系列迄今为止最全面的实验——这些实验最初旨在记录人类偏见。我们记录了 LLM 行为中的系统性模式。在基于偏好的任务中，随着模型变得更先进或规模更大，其响应变得更像人类；而在基于信念的任务中，先进的大规模模型经常生成理性的响应。提示 LLM 做出理性决策可以减少偏见。

## Abstract
Do generative AI models, particularly large language models (LLMs), exhibit systematic behavioral biases in economic and financial decisions? If so, how can these biases be mitigated? Drawing on the cognitive psychology and experimental economics literatures, we conduct the most comprehensive set of experiments to date$-$originally designed to document human biases$-$on prominent LLM families across model versions and scales. We document systematic patterns in LLM behavior. In preference-based tasks, responses become more human-like as models become more advanced or larger, while in belief-based tasks, advanced large-scale models frequently generate rational responses. Prompting LLMs to make rational decisions reduces biases.

#### 1. 核心问题与整体含义
随着生成式 AI 深度介入金融服务、风险评估及科研实验，其决策的可靠性直接影响到社会经济系统的运行效率。本研究的核心动机在于：**验证 AI 是否继承了人类的系统性行为偏见，并探索如何通过优化指令来最大化其决策的理性效用。** 研究通过引入“AI 行为经济学”这一概念，旨在评估 LLM 作为新型经济代理人的行为特征，确保其在替代或辅助人类决策时能够提供正向的社会价值。

#### 2. 方法论
研究采用了跨学科的实验方法，核心思想是将 LLM 视为心理学实验的受试者：
*   **认知心理学进路：** 选取了 16 个经典实验，涵盖“偏好心理学”（如前景理论、损失厌恶、时间偏好）和“信念心理学”（如基率忽视、赌徒谬误、过度自信）。
*   **实验经济学进路：** 模拟真实经济场景，包括基于自回归过程的变量预测任务（考察信念偏见）和基于股票价格轨迹的投资决策任务（考察偏好偏见）。
*   **提示词工程（Prompt Design）：** 设计了标准化的 API 提示词，要求模型以 JSON 格式输出决策结果、置信度、解释及推理类型（直觉型 A 或分析型 B），以确保数据的可量化分析。

#### 3. 实验设计
*   **实验对象：** 涵盖 4 个主流家族的 12 个模型，包括 OpenAI (GPT-3.5/4/4o)、Anthropic (Claude 2/3 Opus/Haiku)、Google (Gemini 1.0/1.5 Pro/Flash) 和 Meta (Llama 2/3)。
*   **对比维度：** 
    *   **纵向对比：** 同一家族的新旧版本（考察技术进步的影响）。
    *   **横向对比：** 同一版本的大规模与小规模模型（考察参数规模的影响）。
*   **Benchmark：** 以冯·诺依曼-摩根斯特恩的“期望效用框架”作为理性基准，同时以认知心理学中记录的人类典型反应作为“类人偏见”基准。

论文未详细披露模型训练阶段的具体算力（如 GPU 型号或时长），因为研究重点在于对现有商业及开源模型的**推理评估**。实验通过 API 调用完成，每项任务进行 100 次独立迭代以消除随机性，总计产生了数万次交互数据。

#### 5. 实验数量与充分性
研究设计非常详尽且具有高度的客观性：
*   **广度：** 覆盖了 12 个模型和 16 类心理学偏见，是目前该领域最全面的研究。

#### 6. 主要结论与发现
*   **偏好与信念的背离：** 在涉及“偏好”的任务中，模型越先进、规模越大，表现得越像人类（即更不理性），这可能归因于 RLHF（人类反馈强化学习）过度对齐了人类的非理性偏好。而在涉及“信念”的任务中，先进模型表现出极高的理性，能够识别统计真相。
*   **规模效应：** 增加模型规模能显著提升逻辑推理和概率计算的准确性（信念理性），但也会强化模型对人类非理性选择模式的模仿（偏好偏见）。
*   **修正的可能性：** 简单的“角色启动”（如提示模型是理性投资者）能适度提升理性，但提供过多的背景知识（如前景理论摘要）反而会因信息过载导致性能下降。

#### 7. 优点
*   **实用性极强：** 为金融机构部署 AI 提供了风险预警，指出了模型在哪些领域可以信任，哪些领域需要人工干预。
*   **理论贡献：** 首次系统性地揭示了 RLHF 可能带来的“副作用”——在追求类人对话的同时，不经意间引入了人类的决策缺陷。
*   **开源贡献：** 建立了公开的实验问题数据库，便于后续研究对新模型进行持续评估。

#### 8. 不足与局限
*   **黑盒限制：** 由于商业模型（如 GPT-4）的训练数据和架构不透明，关于 RLHF 导致偏见的结论主要基于推测。
