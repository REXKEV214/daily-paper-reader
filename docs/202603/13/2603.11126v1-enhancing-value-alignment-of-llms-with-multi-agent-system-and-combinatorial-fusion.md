## 摘要
使大语言模型（LLMs）与人类价值观对齐是确保其可靠和安全部署的核心挑战。虽然现有方法（如基于人类反馈的强化学习 RLHF 及其变体）提高了对齐效果，但它们通常依赖于单一评估者或定义狭窄的奖励信号，限制了其捕捉伦理多元主义的能力。在这项工作中，我们提出了基于组合融合分析的价值对齐系统（VAS-CFA），这是一个实现多智能体融合对齐的框架。它实例化了多个道德智能体，每个智能体都经过微调以代表不同的规范视角，并使用 CFA 通过基于排名和分数的聚合来融合它们的输出。这种设计利用了智能体之间的认知多样性，以减轻多个智能体之间的冲突和冗余，从而产生更好地反映人类价值观的响应。实证评估表明，VAS-CFA 在标准指标上优于单智能体基准和先前的聚合方法，表明多智能体融合为推进 LLMs 的价值对齐提供了一种稳健且有效的机制。

## Abstract
Aligning large language models (LLMs) with human values is a central challenge for ensuring trustworthy and safe deployment. While existing methods such as Reinforcement Learning from Human Feedback (RLHF) and its variants have improved alignment, they often rely on a single evaluator or narrowly defined reward signals, limiting their ability to capture ethical pluralism. In this work, we propose the Value Alignment System using Combinatorial Fusion Analysis (VAS-CFA), a framework that operationalizes multi-agent fusion alignment. It instantiates multiple moral agents, each fine-tuned to represent a distinct normative perspective, and fuses their outputs using CFA with both rank- and score-based aggregation. This design leverages cognitive diversity, between agents, to mitigate conflicts and redundancies across multiple agents, producing responses that better reflect human values. Empirical evaluation demonstrates that VAS-CFA outperforms both single agent baselines and prior aggregation approaches on standard metrics, showing that multi-agent fusion provides a robust and effective mechanism for advancing value alignment in LLMs.

*   **核心问题**：现有的 LLM 价值对齐方法（如 RLHF、DPO）通常依赖单一的评估者或奖励模型，难以捕捉人类价值观的“伦理多元主义”（Ethical Pluralism）。
*   **研究背景**：预训练模型常产生不实或不安全的输出。虽然现有方法有所改进，但往往导致目标过于狭隘，无法处理不同道德准则之间的冲突与冗余。本文旨在通过多智能体协作和数学融合框架，使模型能够同时兼顾多种道德视角。

#### 2. 方法论
核心思想是构建一个名为 **VAS-CFA** 的框架，通过实例化多个具备不同道德倾向的智能体，并利用**组合融合分析（CFA）**技术进行决策集成。
*   **多智能体微调**：基于 `Pythia-12B` 模型，利用 **DPO (Direct Preference Optimization)** 和 **QLoRA** 技术，在道德诚信语料库（MIC）上微调出 5 个专家智能体，分别代表：权威（Authority）、关爱（Care）、公平（Fairness）、忠诚（Loyalty）和圣洁（Sanctity）。
*   **道德单元分解**：使用 GPT-4 将智能体的长响应分解为独立的“道德单元”（Moral Units），每个单元仅包含一个核心道德主张，以避免语义冲突。
*   **道德分类与评分**：训练一个基于逻辑回归的分类器，为每个道德单元在上述 5 个维度上进行评分。
*   **组合融合分析 (CFA)**：
    *   计算**认知多样性 (Cognitive Diversity, CD)**：衡量不同智能体评分系统之间的差异。
    *   计算**多样性强度 (Diversity Strength, DS)**：作为融合时的权重。
    *   **四种聚合策略**：包括平均得分组合 (ASC)、基于 DS 的加权得分组合 (WSCDS)、平均排名组合 (ARC) 以及基于 DS 的加权排名组合 (WRCDS)。
*   **改写生成**：选择 CFA 筛选出的最优单元，通过 Paraphraser 改写为最终的完整回答。

*   **数据集**：使用 **Moral Integrity Corpus (MIC)**，包含 113.8K 个提示-响应对，具有丰富的人类伦理标注。
*   **Benchmark 与对比方法**：
    *   **基准 1**：5 个单一价值的道德智能体（A-E）。
    *   **基准 2**：原始聚合（Raw Aggregation，直接合并响应）。
    *   **基准 3**：先前的 SOTA 方法，如 CVA-GS 和 CVA-GS-DYN（上下文价值对齐）。
*   **评估指标**：ROUGE-L F1 分数和 BERTScore F1 分数。

    *   时长：每个智能体微调 1 个 epoch。

#### 6. 主要结论与发现
*   **多智能体优于单智能体**：VAS-CFA 在 ROUGE-L 和 BERTScore 上均显著超过了任何单一维度的专家智能体。
