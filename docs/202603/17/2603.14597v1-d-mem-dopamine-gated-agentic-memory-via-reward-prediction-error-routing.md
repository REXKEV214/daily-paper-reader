## 摘要
自主大语言模型（LLM）智能体需要结构化的长期记忆，但目前的“追加与演化”系统（如 A-MEM）面临 O(N^2) 的写入延迟和过高的 Token 成本。我们提出了 D-MEM（多巴胺门控智能体记忆），这是一种受生物启发的架构，通过基于奖励预测误差（RPE）的快/慢路由系统，将短期交互与认知重构解耦。一个轻量级的评论家路由器（Critic Router）评估刺激的“惊喜度”（Surprise）和“效用”（Utility）。常规的、低 RPE 的输入会被绕过或缓存在 O(1) 的快速访问缓冲区中。相反，高 RPE 输入（如事实矛盾或偏好转变）会触发“多巴胺”信号，激活 O(N) 记忆演化流水线，以重塑智能体的知识图谱。为了在现实条件下评估性能，我们引入了 LoCoMo-Noise 基准测试，该基准在长期会话中注入受控的对话噪声。评估结果表明，D-MEM 减少了超过 80% 的 Token 消耗，消除了 O(N^2) 瓶颈，并在多跳推理和对抗韧性方面优于基准模型。通过选择性地对认知重构进行门控，D-MEM 为终身智能体记忆提供了一个可扩展且具有成本效益的基础。

## Abstract
Autonomous LLM agents require structured long-term memory, yet current "append-and-evolve" systems like A-MEM face O(N^2) write-latency and excessive token costs. We introduce D-MEM (Dopamine-Gated Agentic Memory), a biologically inspired architecture that decouples short-term interaction from cognitive restructuring via a Fast/Slow routing system based on Reward Prediction Error (RPE). A lightweight Critic Router evaluates stimuli for Surprise and Utility. Routine, low-RPE inputs are bypassed or cached in an O(1) fast-access buffer. Conversely, high-RPE inputs, such as factual contradictions or preference shifts, trigger a "dopamine" signal, activating the O(N) memory evolution pipeline to reshape the agent's knowledge graph. To evaluate performance under realistic conditions, we introduce the LoCoMo-Noise benchmark, which injects controlled conversational noise into long-term sessions. Evaluations demonstrate that D-MEM reduces token consumption by over 80%, eliminates O(N^2) bottlenecks, and outperforms baselines in multi-hop reasoning and adversarial resilience. By selectively gating cognitive restructuring, D-MEM provides a scalable, cost-efficient foundation for lifelong agentic memory.

随着自主大语言模型（LLM）智能体向长期、持久化交互发展，**智能体记忆（Agentic Memory）**成为核心。现有的先进框架（如 A-MEM）采用“动态演化知识图谱”的方案，虽然保证了记忆的一致性，但存在严重的**扩展性瓶颈**：
*   **计算冗余**：系统对每一句对话（无论是否有意义）都执行复杂的知识提取和全局记忆演化，导致计算复杂度随交互增长呈 $O(N^2)$ 爆发。
*   **成本与延迟**：高昂的 API Token 消耗和写入延迟使其难以应用于实时生产环境。

#### 2. 方法论：核心思想与关键技术
D-MEM 受生物学中哺乳动物大脑**腹侧被盖区（VTA）**的多巴胺释放机制启发，提出了一种**快/慢路由架构**，将短期交互与长期认知重构解耦。

*   **核心思想**：并非所有信息都值得存入长期记忆。只有当输入产生较高的**智能体奖励预测误差（Agentic RPE）**时，才触发昂贵的记忆重构。
*   **关键技术细节**：
    *   **Agentic RPE 计算**：由“语义惊喜度（Surprise）”和“长效效用（Utility）”共同决定。
        *   **Surprise**：通过计算当前输入与已有记忆的余弦相似度，并进行 Z-score 标准化来克服嵌入向量的各向异性。
        *   **Utility**：利用轻量级 LLM 判定信息是瞬时的（Transient）、短期的还是持久的（Persistent）。
        *   **公式**：$RPE(x_t) = \min(1.0, \mathbb{I}(Utility \ge \tau) \cdot [Utility \times (Surprise + \beta)])$。
    *   **三层分级路由**：
        1.  **SKIP (低 RPE)**：直接跳过，存入 $O(1)$ 的影子缓冲区（Shadow Buffer）。
        2.  **CONSTRUCT_ONLY (中 RPE)**：仅生成原子节点存入短期缓冲区，不进行图关联。
        3.  **FULL_EVOLUTION (高 RPE)**：触发“多巴胺释放”，执行 $O(N)$ 的深度记忆演化和全局图谱更新。
    *   **零成本检索增强**：结合 BM25 稀疏检索与向量检索（RRF 融合），并利用影子缓冲区应对对抗性查询。
