## 摘要
基于大语言模型（LLM）的智能体在视觉语言导航（VLN）任务中展现了令人印象深刻的零样本性能。然而，大多数零样本方法主要依赖闭源 LLM 作为导航器，这面临着高昂的 token 成本和潜在的数据泄露风险。最近的研究尝试通过结合开源 LLM 与时空思维链（CoT）框架来解决这一问题，但与闭源模型相比仍有较大差距。在这项工作中，我们通过对导航过程的详细分析，发现了一个关键问题：导航失忆症（Navigation Amnesia）。这一问题导致了导航失败，并扩大了开源与闭源方法之间的差距。为了解决这一问题，我们提出了 HiMemVLN，它将分层记忆系统集成到多模态大模型中，以增强视觉感知召回和长期定位，从而缓解失忆问题并提高智能体的导航性能。在模拟和真实环境中的广泛实验表明，HiMemVLN 的性能几乎是现有开源最先进方法的两倍。代码可在 https://github.com/lvkailin0118/HiMemVLN 获取。

## Abstract
LLM-based agents have demonstrated impressive zero-shot performance in vision-language navigation (VLN) tasks. However, most zero-shot methods primarily rely on closed-source LLMs as navigators, which face challenges related to high token costs and potential data leakage risks. Recent efforts have attempted to address this by using open-source LLMs combined with a spatiotemporal CoT framework, but they still fall far short compared to closed-source models. In this work, we identify a critical issue, Navigation Amnesia, through a detailed analysis of the navigation process. This issue leads to navigation failures and amplifies the gap between open-source and closed-source methods. To address this, we propose HiMemVLN, which incorporates a Hierarchical Memory System into a multimodal large model to enhance visual perception recall and long-term localization, mitigating the amnesia issue and improving the agent's navigation performance. Extensive experiments in both simulated and real-world environments demonstrate that HiMemVLN achieves nearly twice the performance of the open-source state-of-the-art method. The code is available at https://github.com/lvkailin0118/HiMemVLN.

### 1. 核心问题与研究背景
*   **研究动机**：视觉语言导航（VLN）要求智能体根据自然语言指令在未知环境中移动。目前最先进的方法依赖闭源大模型（如 GPT-4o），但这存在高昂的 API 成本、隐私泄露风险以及网络延迟问题。
*   **核心问题**：开源大模型在零样本（Zero-shot）导航中表现远逊于闭源模型。作者通过分析发现，开源模型存在**“导航失忆症”（Navigation Amnesia）**：
    *   **短期失忆**：无法识别已访问的相似场景，导致陷入局部死循环或重复探索。
    *   **长期失忆**：随着步骤增加，模型会忘记全局指令约束，导致决策偏离预定路线。

### 2. 方法论：HiMemVLN 框架
受神经科学中“海马体记忆索引理论”启发，论文提出了一个分层记忆系统，将导航过程转化为“感知-记忆-推理-执行”的闭环：
*   **短期定位器系统（Short-Term Localer）**：
    *   **视觉图记忆**：构建一个在线更新的图，节点存储位置的 CLIP 视觉特征、访问次数和时间戳。
    *   **重定位机制**：通过余弦相似度匹配当前观察与历史节点。若相似度高于阈值，则判定为“重访”，并利用动量更新节点特征。
    *   **软约束引导**：将重访统计信息转化为文本提示，给 MLLM 提供“避开已探索区域”的建议。
*   **长期全局器系统（Long-Term Globaler）**：
    *   **全局模式提取**：在任务开始时提取主方向、目标地标和导航模式。
    *   **来源追踪（Origin Tracking）**：实时维护一个“来自何处”的变量，增强智能体的方向感。
    *   **反射推理**：在每一步决策时，强制模型回顾全局目标和来源方向，校准局部动作。
*   **航点预测与感知**：使用预训练的航点预测器生成候选动作，并结合 RAM（标签识别）和 SpatialBot（空间理解）将视觉信息转化为结构化文本。

*   **实验规模**：模拟环境进行了 100 个 Episode 的评测；真实环境设计了 20 条涵盖简单到复杂的导航指令。

*   **性能飞跃**：HiMemVLN 在 R2R-CE 上的成功率（SR）达到 30%，相比之前最强的开源方法（16%）提升了近一倍。

*   **可靠性提升**：分层记忆系统有效解决了开源模型在长距离导航中的“死循环”和“目标漂移”问题。

### 7. 优点与亮点
*   **认知启发式设计**：将抽象的神经科学理论成功转化为可落地的分层记忆工程架构。
*   **隐私与成本优化**：通过增强开源模型的能力，实现了无需上传云端、低成本且隐私安全的具身智能部署。
*   **即插即用**：该框架不依赖特定模型，可广泛适配各种开源多模态大模型。

*   **真实环境样本量**：真实世界的测试指令（20条）相对较少，可能无法完全覆盖极端复杂的动态环境。

*   **模块依赖**：导航性能在很大程度上仍受限于前端“航点预测器”和“视觉描述模型”的准确性。
