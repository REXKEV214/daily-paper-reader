## 摘要
随着多模态语言模型的飞速发展，在日常生活中辅助人类的 AI 助手正变得日益可行。一个关键挑战在于如何克服这些模型的通用性，以提供个性化的体验。现有的视觉语言大模型个性化方法通常依赖于额外的训练阶段，这限制了通用性和可扩展性；或者依赖于带有外部预训练模块的工程化流水线，这阻碍了部署效率。在这项工作中，我们提出了一种高效的个性化方法，利用模型捕获个性化概念的固有能力。具体而言，我们利用模型的内部注意力机制，提取主要代表目标概念的视觉 token。这些 token 作为该特定概念的记忆，使模型能够在测试图像中出现该概念时对其进行召回和描述。我们在包括单概念、多概念和视频个性化在内的各种个性化设置下，对我们的方法和 SOTA 方法进行了全面且统一的评估，结果表明我们的方法在极小的个性化开销下实现了显著的性能提升。

## Abstract
AI assistants that support humans in daily life are becoming increasingly feasible, driven by the rapid advancements in multimodal language models. A key challenge lies in overcoming the generic nature of these models to deliver personalized experiences. Existing approaches to personalizing large vision language models often rely on additional training stages, which limit generality and scalability, or on engineered pipelines with external pre-trained modules, which hinder deployment efficiency. In this work, we propose an efficient personalization method that leverages the model's inherent ability to capture personalized concepts. Specifically, we extract visual tokens that predominantly represent the target concept by utilizing the model's internal attention mechanisms. These tokens serve as a memory of that specific concept, enabling the model to recall and describe it when it appears in test images. We conduct a comprehensive and unified evaluation of our approach and SOTA methods across various personalization settings including single-concept, multi-concept, and video personalization, demonstrating strong performance gains with minimal personalization overhead.

*   **现有挑战**：
    *   **微调类方法**：需要针对每个新概念进行测试时微调（Test-time Fine-tuning），扩展性差且计算成本高。

*   **核心目标**：提出一种**无需训练（Training-free）**、**模型无关（Model-agnostic）**且**高效**的个性化框架，利用模型自身的注意力机制来构建“概念记忆”。

### 2. 论文提出的方法论：Ego
Ego（Embedding-Guided Personalization）的核心思想是：**利用 LVLM 内部的注意力机制，从参考图像中提取最具代表性的视觉 Token，并将其作为“软提示（Soft Prompts）”存储在上下文记忆中。**

*   **关键技术流程**：
    1.  **概念引入（Concept Introduction）**：输入参考图像，让模型生成描述该主体的关键词（Keywords）。
    2.  **注意力引导的 Token 提取**：
        *   计算模型内部 LLM 层的交叉注意力图（Cross-attention maps）。
        *   分析关键词 Token 对视觉 Token 的注意力权重，识别出模型认为最能代表该主体的视觉区域。
        *   **公式逻辑**：通过对多个注意力头和特定层（通过自动校准选定）的得分进行聚合，计算每个视觉 Token 的重要性得分 $I_j$。
    3.  **动态记忆大小（Dynamic Memory Size）**：让模型估计主体在图像中的占比（$\alpha_c$），据此动态决定提取的 Token 数量 $K_c$，从而过滤背景噪声并节省显存。
    4.  **推理阶段（Inference）**：将提取的视觉 Token（已处于 LLM 嵌入空间）直接作为上下文输入，配合概念名称，引导模型在测试图中识别并推理相关概念。

### 6. 论文的主要结论与发现
*   **性能领先**：Ego 在多概念识别和视频 VQA 任务上显著优于现有的训练类和免训练类方法。例如，在 This-is-my 多概念任务中，F1 分数比 RAP 高出约 12%。

*   **多概念优势**：传统方法在多概念场景下容易出现混淆或召回率骤降，而 Ego 通过精准的 Token 提取，能够同时保持多个实体的辨识度。
*   **视频能力**：Ego 无需修改即可直接应用于视频流，通过在时间维度上维持概念记忆实现一致性推理。

### 7. 优点（亮点）
*   **完全免训练**：不需要任何参数更新，也不需要针对特定任务的微调。
*   **内部一致性**：不依赖外部的检测或分割模型，而是利用模型“自己的眼光”（内部注意力）来定义物体，减少了模态对齐误差。
*   **高度可扩展**：支持单图或多图参考，支持多概念共存，且推理速度快。
*   **统一框架**：一个方法同时解决了识别、问答、描述和视频理解四类任务。

### 8. 不足与局限
*   **模型依赖性**：Ego 依赖于基础模型强大的上下文学习（ICL）和视觉理解能力。在较弱或较旧的模型（如参数量极小的模型）上，其性能会受限。
*   **提示词敏感性**：作为一种 Training-free 方法，它对初始引入概念时的 Prompt 比较敏感，需要针对不同底座模型进行简单的 Prompt 调整。
*   **极端遮挡**：虽然在常规场景下表现稳健，但在极端遮挡或极小目标的情况下，注意力机制可能无法精准定位所有关键特征。
